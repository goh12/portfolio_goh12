Web Application

This project consists of two seperate repositories. The first one is a JSON web service. The web service is for a (fictional) library. Stored data consists of users, categories, books and user's read books. The data is stored using Postgres on a Heroku server. The second repository is a front end for the web service. The front end was built in React. Users can search for books, mark them as read and review them. This project was a group project for the course Web Programming 2 in the spring semester 2018.
